Vozen Helper

Política de Privacidade

Última atualização: 15 de julho de 2026

O Vozen Helper é um bot de moderação e comunidade que opera num único servidor Discord privado. Esta página explica que dados o bot trata, para quê, durante quanto tempo, e como podes exercer os teus direitos ao abrigo do Regulamento Geral sobre a Proteção de Dados (RGPD).

1. Quem é responsável

O responsável pelo tratamento dos dados é o administrador do servidor onde o Vozen Helper está instalado. Para qualquer questão de privacidade, contacta-o diretamente através do Discord (mensagem direta ou um ticket no servidor).

2. Que dados tratamos

O bot trata sobretudo identificadores do Discord (o teu ID de utilizador, um número público que a plataforma te atribui). Em concreto:

  • Moderação: registos de avisos, sanções, notas de staff, infrações e quarentenas — associados ao teu ID.
  • Comunidade (opcionais, criados por ti): sugestões e votos, estado AFK, lembretes, participação em giveaways e o teu dia e mês de aniversário (nunca o ano) se o registares.
  • Atividade: XP/nível de participação e tickets de suporte que abras.
  • Estatísticas: contagens diárias agregadas do servidor (mensagens, entradas, saídas) — sem te identificar.

Não recolhemos o teu nome civil, email, número de telefone, morada, endereço IP ou o conteúdo geral das tuas conversas. O bot não lê nem guarda mensagens para além do estritamente necessário às funções acima.

3. Com que fundamento

  • Interesse legítimo em manter o servidor seguro e funcional — para os dados de moderação, XP, tickets e estatísticas.
  • Consentimento / ato voluntário — para tudo o que és tu a criar (aniversário, AFK, lembretes, sugestões, giveaways). Podes retirá-lo a qualquer momento.

4. Durante quanto tempo

  • Registos de moderação: até 2 anos (infrações e quarentenas até 1 ano).
  • XP/nível: apagado quando sais do servidor.
  • Dados opcionais (aniversário, AFK, sugestões, giveaways): até os apagares, saíres do servidor, ou o prazo do recurso terminar.
  • Estatísticas agregadas: 1 ano.
  • Registos técnicos da API do painel: até 30 dias.

5. Com quem partilhamos

Os dados não são vendidos nem cedidos para marketing. São processados por serviços estritamente necessários ao funcionamento:

  • Discord — a plataforma onde tudo acontece (política própria da Discord).
  • GitHub Pages — aloja este site estático.
  • Cloudflare — encaminha os pedidos do painel para o servidor, via túnel.
  • O servidor (VPS) que corre o bot e guarda a base de dados.

6. O painel de administração

O painel web (esta página faz parte dele) é acessível a uma única conta: a do administrador. Para manter a sessão iniciada, usa um cookie (vh_session) e o armazenamento de sessão do navegador (vh_sess), ambos estritamente necessários e sem qualquer finalidade de rastreio ou publicidade — por isso não há banner de cookies. A sessão expira sozinha e é apagada ao terminar sessão. O token de login do Discord é usado apenas para confirmar a identidade e não é guardado.

7. Os teus direitos

Tens o direito de aceder, corrigir, apagar e limitar o tratamento dos teus dados. Podes exercê-los diretamente no servidor:

  • /privacidade dados — recebes, por mensagem direta, tudo o que o bot tem associado a ti.
  • /privacidade apagar — apaga os teus dados voluntários.

Os registos de moderação podem ser mantidos mesmo após um pedido de apagamento, quando forem necessários para a segurança do servidor ou para a defesa de direitos (art. 17.º, n.º 3 do RGPD). Nesse caso, o bot informa-te de forma clara. Se não concordares com o tratamento, podes apresentar reclamação à Comissão Nacional de Proteção de Dados (CNPD).

8. Alterações

Esta política pode ser atualizada. A data no topo indica a última revisão.
